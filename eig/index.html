EIGSH Command Reference

📂 `Account` Commands

abusetool <cpanel|email|http|exim|script|cron> <enable|disable> <ticket-id> <domain> [server] <no_customer_notification>

→ Enable/disable services like cPanel/sites/emails/crons for an account.

backupdu <domain|ipaddress|server>

→ Prints disk usage of a Dedi/VPS's cPanel backup directory.

backupgen <user> <server> <daily|weekly|monthly|bimonthly> <case-id> <email-id>

→ Generates backups for a particular user. Email ID must be a newfold one.

bui <domain> [server]

→ Display domain/account billing information.

checkcancel <user> <server>

→ Displays last cPanel and FTP login times for an account.

checkrestrictions <domain> [server]

→ Shows all montool restrictions on an account.

cloudsearch <cloudsite username>

→ Looks up information for a Cloudsites account.

countproc <domain> [server]

→ Check the current number of processes for a user.

cplogtable <domain> <server>

→ Summarized cPanel log as a table.

dcbackuplist <user> <server> [brand]

→ List datacycle backups for a user.

dcbackuprestore <user> <daily/weekly/monthly/latam> <server>

→ Restore datacycle backups for a user.

dcfulldatarestore <user> <daily/weekly/monthly> <server> [--archived] [--notify=]

→ Restores home directory and all MySQL databases.

dclistfiles <user> <daily/weekly/monthly> <server> <path> [--archived] [--partition=]

→ Shows files/directories from backup at given path.

dcrestoremysqldb <user> <server> <daily/weekly/biweekly/monthly> <dbname> [--archived]

→ Restores single MySQL database for a user.

dcrestorepath <user> <server> <daily/weekly/biweekly/monthly> <path> [...]

→ Restore file/directory from datacycle backups.

dcv <domain>

→ Returns possible DCV email addresses for a domain.

dcviewfile <user> <server> <daily/weekly/biweekly/monthly> <file path>

→ Print contents of a file from backup.

dedireset <IP>

→ Reset the root password for a dedicated server.

domlist <user> <server>

→ Lists domains for a user.

forcessl <domain> <server>

→ Add HTTPS redirection code in `.htaccess`.

gatorlookup <email|domain>

→ Checks if a domain/email is a GATOR customer.

inodes <server> <path>

→ Show inode breakdown for a path.

killproc <domain> <server>

→ Kill all current processes for a user.

maildirsize <user> <server>

→ Generate default maildir size.

master_check <server> <domain> <ip> <option>

→ Master diagnostics with various options like `--all`, `--disk`, `--mail`, `--whois`, etc.

ncdu <domain> [server]

→ Generate explorable directory tree by size for a user.

openport <domain|server> <port> <ticket-id|chat-id|genesys-id>

→ Opens a port outbound globally on a server.

overselltemp <domain> [server]

→ Temporarily allow reseller to oversell for 24 hours.

restrict <domain> [server]

→ Apply htpasswd restriction to a customer account.

resusage <reseller domain> <server>

→ Print disk and bandwidth usage for a reseller.

resuspend <domain> <reason> <server>

→ Suspend a reseller and all sub-accounts.

reunsuspend <domain> <server>

→ Unsuspend a reseller and all sub-accounts.

searchcpanel <domain> [server]

→ Search cPanel access logs.

skipbackup <user> <server> <vps|next|db>

→ Skip next backup for an account.

sloptout <domain> [server]

→ Opt user out from regular SiteLock scanning.

suspbackup <user> <ticket-id> <server>

→ Creates a user-accessible backup.

suspend <domain> <reason> <server>

→ Suspend a user. Reason must not contain spaces.

testlogin <server|ip|domain> <user> <password>

→ Test password of the account.

toswhitelist <domain> <customer ip> [server]

→ Whitelist customer IP against TOS restrictions.

ui <domain> [server]

→ Display domain/account information.

unlimit <domain> [server]

→ Remove limits on an account.

unrestrict <domain> [server]

→ Remove htpasswd restriction on customer account.

unsuspend <domain> <server>

→ Unsuspend a user.

🔥 `Apache` Commands

apacherestart <domain> <server>

→ Restarts Apache for a domain.

awstats <primary domain> <server> [number of top results]

→ Display top traffic from AWStats. 0 = this month.

cachectl <user> <on|off|status> <server> <reason>

→ Enable/disable/status of cache for a user.

checkautossl <user> <server>

→ Check AutoSSL configuration for a user.

checksites <domain|user> <-r|-u> <server>

→ Check domains under a user or reseller for issues.

createinfo <domain> <server>

→ Create a PHP info page in user's public_html.

createini <domain> <server>

→ Create editable `php.ini` in user's home dir.

disableuserdir <user> <server> [1]

→ Disable userdir access. Optional 1 for immediate apply.

edithtaccess <domain> <server>

→ Edit `.htaccess` file in user's account.

editini <domain> <server>

→ Find and edit all `php.ini` files for domain.

listphphandlers <domain> <server>

→ List all .htaccess, php.info, user.ini files.

phpallow <domain> <server> <52|53|54|55|56|70|71|all>

→ Allow user to choose deprecated PHP versions.

purgecacheuser <server> <user>

→ Purge cache on all domains for a user.

pwrestrict <server> <user> <-b|-r>

→ Run pwrestrict on a LATAM shared account.

scoreboard <domain> <server>

→ Check Apache concurrent connections.

sitereport <domain> <server>

→ Check and report common issues for an account.

sslinfo <domain> [server]

→ Show SSL info for all domains under an account.

uf <domain> <server>

→ Display all vhosts matching the domain.

whitelist <domain> <add|del> <server> <rule id>

→ Add or remove mod_security rule whitelist for a domain.

🗂️ `Archivedlog` Commands

archivedlog <server> <domain> <cpaccess|loginlog|ftp|ssh|mail|mailaccess>

→ Fetch archived logs of specified type for domain/server.

`bkprestore` Commands

bkprestore [server] [User] [Ticket]

→ Initiates full backup restore.

`cPanel` Commands

acctsummary <user> <server>

→ Summarizes account info like usage, domains, settings.

bandwidth <user|domain> <server>

→ Shows bandwidth usage.

cpconftool <domain> <server>

→ Repairs cPanel configuration.

cpuserlogins <user> <server>

→ Retrieves recent login activity for a user.

emailauth <domain> <server>

→ Displays DKIM/SPF status.

enableapi <user> <server>

→ Enables API token access.

fixauthnlinks <user> <server>

→ Fixes broken links in email authentication records.

fixcpmysql <user> <server>

→ Repairs MySQL-related permissions/files for cpanel user.

mailperm <user> <server>

→ Resets mail folder/file permissions.

quota <user> <server>

→ Checks and repairs quota for the user.

rebuildquotas <server>

→ Recalculates disk quotas across accounts.

suspendpkg <package> <server>

→ Suspends a package.

upcp <server>

→ Runs cPanel update script.

`DNS` Commands

checkdns <domain> <server>

→ Performs DNS consistency check.

dnscompare <domain> <server1> <server2>

→ Compares DNS records between two servers.

fixdns <domain> <server>

→ Repairs broken DNS entries.

setrdns <ip> <domain>

→ Sets reverse DNS entry.

syncdns <domain> <server1> <server2>

→ Syncs DNS zones.

`Email` Commands

checksmtp <domain> <server>

→ Tests SMTP sending capabilities.

clearqueue <server>

→ Empties mail queue.

disablerewrite <user> <server>

→ Disables sender rewriting.

dkimrepair <domain> <server>

→ Repairs DKIM record.

eximlog <domain|email> <server>

→ Shows email delivery log.

fixspfrecord <domain> <server>

→ Repairs SPF DNS entry.

mailq <server>

→ Shows current mail queue.

mailtrace <email> <server>

→ Traces mail path.

spfrecord <domain> <server>

→ Displays SPF record.

`FTP` Commands

ftpinfo <domain> <server>

→ Displays FTP account info.

ftpreset <user> <server>

→ Resets FTP access/config.

ftpstatus <server>

→ Shows FTP service status.

`getdnszone` Commands

getdnszone <domain>

→ Displays full DNS zone.

`iskilled` Commands

iskilled <domain> <server>

→ Checks if domain was previously suspended/killed.

`malware_remove` Commands

malware_remove <domain> <server>

→ Initiates malware scan and clean.

`Misc` Commands

checkports <server>

→ Lists open/filtered ports.

checkuptime <server>

→ Displays system uptime.

digcheck <domain>

→ Performs DNS dig.

portscan <server>

→ Scans for open ports.

syslog <server>

→ Displays system log entries.

version <server>

→ Displays OS/cPanel/software versions.
